Raptor 常见问题与解答

🧃 支持哪些第三方下载器?

本程序支持的第三方下载器如下：

  • IDM (仅 Windows)
  • FDM

PS: IDM 需要您拥有许可信息。其次对于单个文件的下载可充分发挥 IDM 等下载器的多线程优势，但对于一次性下载多个文件且可能含有目录的情况下如果选择 IDM 或者 FDM 会频繁弹出单独的窗体，所以如有此类传输需求建议您不要选择第三方引擎进行下载。

🥤 支持哪些播放器?

本程序支持的播放器如下：

  • MPV
  • VLC
  • PotPlayer

PS: 从 2024.2.0 版本起不再携带上述第三方播放器的二进制分发版本。需要您自行下载并解压至 Raptor 所在路径的 Store。或者您也可以在 [设置 - 播放] 设置为您计算机中已经存在的上述第三方播放器的路径

🥝 如何切换播放器?

依次点击左侧导航栏中的 [设置 - 播放]，即可看到所有 Raptor 集成的播放引擎。双击您想切换引擎的 名称 列即可切换。

🍾 如何切换下载引擎或者播放引擎的其他版本?

依次点击左侧导航栏中的 [设置 - 下载|播放]，即可看到所有 Raptor 集成的引擎。双击您想切换引擎的 路径 列，输入您的可执行文件的路径即可。

🐾 为什么我双击播放的明明是一个音频/视频，但是却无法唤起播放器?

如出现该类情况，可能是您的播放器没有正确的配置，或者您上传的音频/视频被阿里云盘以二进制流的 Mime 类型存储。而双击播放动作则是依据 Mime 类型来识别是否属于音频/视频，从而判断是否应该唤起播放器。

🍍 如何设置每个下载任务的并行数量? 为什么我无法设置下载文件时的并行参数?

理论上并行数字越大传输速度会越快。但是如果过大会对服务器造成压力，因此我们建议您如果不是尊贵会员身份切仍然想跑满下行带宽，建议设置为 16 或 32。该参数仅在您不是尊贵会员身份时才会启用。

🤨 什么时间提供 Linux 和 MacOS 版本?

适配 Linux 和 MacOS 版本已在计划当中。 在 2024.3.0 中会开始适配，最晚在 2024.4.0 中可用。

🤨 什么时候可以预览图片?

该功能将在 2024.3.0 中提供。

🍀 我确定我所选择的文件是音频/视频，如何播放?

如果您认为您所选的文件可以播放，请按 Ctrl + Shift + Enter 可强制唤起播放器。

🦋 如何创建多级文件夹?

在创建文件夹时，输入格式 文件夹01/文件夹02 即可创建多级文件夹。

🧀 在未来的版本中，是否会接入阿里云盘的 OpenAPI?

在 2024.1.0 版本初始发行时，已收到阿里云盘开放平台产品经理的邀请邮件。后续会在额外适配一版（名字还未确定） OpenAPI。Raptor 将仍然使用 WebAPI。

🍧 为什么我修改了配置，但是不起作用?

在您修改某些配置项，如 [主题] 。只需重启 Raptor 即可生效。

🍷 使用 Raptor 会不会封禁账号?

依据阿里云盘官方的回复。导致账号封禁的唯一原因是您将下载地址分享给了其他人，导致同一个下载地址有不同的 IP 进行请求。因此请不要将下载地址分享给他人。

🥥 如何自定义界面上的文案?

打开 Raptor.db 中的 CopyWriter 表，即可随意修改。
